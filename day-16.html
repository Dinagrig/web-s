Flask

Для начала нужно открыть хотя бы пустую страницу. Для этого используем Flask. Его нужно импортнуть и заключить в переменную:

        from flask import Flask
        app = Flask(__name__)

Саму страницу откроем следующим образом:

        @app.route('/example (навигационное название страницы)')

В скобках и кавычках прописывается ссылка на страницу, которая будет видна в строке поиска.

Сюда можно/нужно добавить функцию, с помощью которой будет запускаться файл с кодом содержимого страницы:

        @app.route('/example (навигационное название страницы)')
        def example():
            return render_template('example.html (Название файла)')
